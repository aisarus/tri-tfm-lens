TRI·TFM Lens — Privacy Policy

Last updated: March 6, 2026

What data does the extension access?

When you click "Evaluate", the extension reads the AI response text on the current page (ChatGPT or Google Gemini only).

Where is the data sent?

The response text is sent to the Google Gemini API (generativelanguage.googleapis.com) for quality evaluation. This is the only external service the extension communicates with. The API call uses YOUR personal API key.

What data is stored?

Only your Gemini API key and evaluation mode preference are stored locally on your device using chrome.storage.local. No data is stored on any external server.

What data is NOT collected?

No analytics, no tracking, no cookies, no user profiles, no browsing history. The extension does not collect, store, or transmit any personal information.

Third parties

The only third party service is the Google Gemini API, governed by Google's own privacy policy.
